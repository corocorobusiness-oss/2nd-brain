# スレタイバンク（本物・コーパス由来 / creative-thread-gen スワップ生成用）

> creative-thread-gen がスレタイを作るときの**本物の土台**。`スレコーパス/*.md` の実スレタイを型ラベル＋字数つきで列挙したもの。
> 🎯**使い方（本物スワップ法）**：スレタイはゼロから書かず、**この中からテーマに近い型を1本選び、人物名／数字／通説ワードだけ差し替える**。記号（「」←【】ｗの個数）・語尾の崩し・リズムは本物のまま固定する。
> 🔁**更新**：コーパス追加時に `スレコーパス/*.md` の `# / ##` 見出しから再抽出→ノイズ除去→型分類（最終抽出 2026-06-17）。
> 📊**総数 61本**。型を偏らせない目安にも使う（本物実測の出現比：セリフ寸劇29／末尾ｗ草21／【朗報悲報】20／疑問14／←12／断定同意10／とかいう4％）。
> ⚠️**禁止語注意**：本物なので一部に禁止語（殺／糞 等＝「脱糞敗走」「利休殺したんや」等）が混じる。**スワップ時は必ず言い換える**（殺→やる/討つ、脱糞→敗走/漏らす）。最終はqa_checkが弾く。

**関連**: [[2chスレ文体パターン]]（型カタログ＋スレタイ生成プロトコル） / `qa_check.py`（スレタイ機械ゲート）
