# 著作権侵害による削除通知

出典: https://support.google.com/youtube/answer/2807622?hl=ja
最終取得: 2026-04-03

---

著作権侵害による削除通知を提出する

著作権で保護されているご自身のコンテンツが無断で YouTube に掲載されている場合は、著作権侵害による削除通知を提出し、法的手続きを行うことができます。動画の場合は、YouTube Studio で提出するか、メール、ファックス、郵便のいずれかで提出できます。動画以外のコンテンツ（チャンネル バナー画像など）については、メール、ファックス、郵便のいずれかにてご提出ください。

提出にあたっての注意事項:

著作権の例外: フェアユース、パブリック ドメイン、または同様の著作権の例外が適用されないかご確認ください。このような確認がお済みかどうかを、YouTube からお尋ねすることがあります。十分なご回答をいただけない場合、著作権侵害による削除通知が無効になり、削除が実行されないことがあります。
スケジュール設定された削除通知: 7 日後に有効になるように、著作権侵害による削除通知をスケジュール設定することができます。この設定を行うことで、アップロードしたユーザーは 7 日間の猶予が与えられ、この期間に対応策をとれば著作権侵害の警告を回避できます。
コピーの防止: YouTube Studio から提出する場合は、同じコンテンツのコピーが今後再アップロードされるのを防止するオプションを選択できます。
共有される情報: 法律上、YouTube ではアップロードしたユーザーと特定の情報を共有することが義務付けられています。個人情報が開示されることについて懸念がある場合は、正式な代理人（弁護士など）がご本人に代わって異議申し立て通知を送信できます。
提出する資格のある方: 著作権侵害による削除通知は、著作権者が提出する必要があります。または、弁護士などの正式な代表者が著作権者に代わって提出することもできます。
虚偽の情報を送信しないでください。不正な書類を提出するなどして、Google の手続きを不正に使用した場合、アカウントの停止や法的問題に発展する可能性があります。
YouTube Studio で提出する

動画の著作権侵害による削除通知は、YouTube Studio から直接提出できます。

著作権侵害による削除通知を提出する

YouTube Studio からこのフォームにアクセスするには:

パソコンまたはモバイル ブラウザで YouTube Studio にログインします。
左側のメニューから [コンテンツ検出]  を選択します。
[新しい削除リクエスト] をクリックします。
メール、ファックス、郵便で提出する

動画や動画以外のコンテンツ（チャンネル バナー画像など）の著作権侵害による削除通知は、メール、ファックス、郵便のいずれかでご提出いただけます。

下記の必要事項を、各セクションを開いて詳細を確認し、すべてメールの本文にご記入の上（添付ファイルとしてではなく）、copyright@youtube.com 宛てに送信してください。FAX または郵送での提出も受け付けています。

1. 連絡先情報
2. 著作権で保護されたコンテンツの説明
3. コンテンツへのリンク
4. 法的な声明
5. 署名
ヒント: 著作権で保護されたコンテンツを YouTube で多数管理している場合は、高度な著作権管理ツールの詳細を確認し、利用資格があるかどうかを確認されることをおすすめします。
