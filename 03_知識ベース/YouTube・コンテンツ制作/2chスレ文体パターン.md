# 2chスレ文体パターン集（オリジナルスレ生成用）

> スクレイピングしたスレが `スレコーパス/` に溜まるたびに、ここのパターンが充実していく。
> オリジナル台本（実スレなし）を作るとき、台本AIはこのファイルを必ず読んで「本物のスレらしさ」を再現する。
> ⚠️ オリジナル生成時も歴史事実はファクトチェック必須（youtube-script-checker）。

**関連**: [[台本執筆ルール]]（離脱データから確定したルール） / `スレコーパス/`（学習素材の実スレ）

## 📐 スレの定番構造（起承転結）

1. **イッチの問題提起**（煽り気味・断定形が伸びる）
   - 「〇〇とかいう△△、過大評価だよな」「〇〇って実際どうなん？」
2. **懐疑・反発レス**（2〜5レス目あたり）
   - 「は？にわかか？」「また〇〇アンチスレか」
3. **知識ニキ登場**（流れが変わる転換点）
   - 「実際は〜やで。史料の〇〇によると…」と長文で本筋の解説
4. **脱線・小ボケ**（張り詰めを緩める。1〜2回）
   - 関係ありそうで微妙にズレた話、現代との比較ネタ
5. **再収束→結論**（イッチ納得 or 言い負かされ、で締まる）
   - 「なんやかんや〇〇は有能やったんやな」

## 🗣 口調・語彙パターン

| 種類 | 例 |
|---|---|
| 語尾（なんJ風） | 〜やで、〜やろ、〜んや、〜やんけ、〜ニキ、〜ンゴ |
| 同意 | これ、ほんこれ、わかる、せやな |
| 反発 | は？、にわか乙、エアプかよ、また始まった |
| 驚き | ファッ!?、マ？、こマ？、草、大草原 |
| 知識ニキの枕 | 「実際は」「マジレスすると」「史実やと」「これ豆な」 |
| 煽り | 〜定期、〜説、ソースは？、はい論破 |

## ✍️ レスのリズム
- 1レス1〜3行が基本。知識ニキの解説レスだけ4〜8行OK
- 短レス（ツッコミ・草）と長レス（解説）を交互に → 単調回避
- 安価（>>1 など）は使いすぎない（動画では音声にしにくい）
- 全レスが標準語はNG。7割なんJ語、3割標準語が自然

## 🚫 オリジナル生成時の禁止事項
- 実在しない「史実」の捏造（諸説あるものは「〜説もある」と明示）
- 現代の特定個人・団体への言及
- NGテーマ（世界史・近代戦争・残虐詳細・性風俗直接表現）
- **禁止ワード（全活用形）**: 死／殺す／バカ・馬鹿／クソ・糞 はチャンネル絶対NG（オーナー指定）。生成時から使わない。死に触れるなら「亡くなる/命を落とす/果てる/遺体/あの世」等に逃がす。「殺された」は「やられた」、グロ描写は概要だけにトーンダウン（[[台本執筆ルール]]）

## 🔥 本物らしさルーブリック（2026-06-15確定・生成時に必ず満たす）
> 最大の失敗パターン＝**「優等生すぎ」**（説明口調が連発・全員が素直に感心・綺麗にオチる）。これを避けるため以下を必ず混ぜる：
1. **懐疑・煽り合い**: 「は？盛りすぎやろ」「ソースは？」「エアプか？」「ぐぐれ」→知識ニキが「いやマジやって」「史料に残っとる」で言い返す摩擦を作る
2. **既出ツッコミ**: 「それ何回も既出やろ」など（※同じ話題を実際に繰り返すのはNG。重複の自白にならないよう1回だけ）
3. **死にレス**: 誰にも拾われない単発レスを1つ（例「ワイの先祖〇〇説」→スルー）
4. **知識ニキへの茶々**: 「知識ニキ落ち着けや、誰も完璧な反論求めてへん」
5. **gdオチ**: 結論を綺麗にまとめすぎない。「諸説ありすぎて草」「結局よう分からんままかい」最後まで認めない逆張りニキ、で締めずに緩く終わる
6. **緩急**: 短い即レス（草／は？／それな）と長文マジレスを交互に。知識ニキ解説が3連続したら間に短い茶々を挟む
- **カッコ**: 引用は「」、セリフ内の引用だけ『』。（白目）（震え声）（遠い目）等の感情表現を適度に
- **自己レビュー必須**: 生成後に「優等生になってないか／同一人物が説明連発してないか／禁止ワード／史実」を自分でチェックして直す

## 📊 学習メモ（コーパスから抽出した傾向）
- 2026-06-15: 縄文・千利休のオリジナル生成で上記ルーブリックを実証。初稿は毎回「優等生すぎ」になり、ザラつき注入で本物感が出た。
- 2026-06-15: コーパスに戦国系10本追加（打線/語り/人物評価/ランキング）。実スレから抽出した具体パターン👇
  - **候補ツッコミ構文**: 打線/ランキング系スレでは「〇〇は？」が大量に湧く（例「前田慶次は？」「楠木正成は？」「墾田永年私財法は？」）。"入ってない有名どころ"を突っ込むのがお約束。創作スレでもこれを5〜8割の頻度で散らすとリアル
  - **緩い言い切り**: 「〇〇やろ」「〇〇やろなぁ」「これで決まりやろ」「川中島は外せないやろなあ」＝断定を避けつつ推す口調
  - **定番煽り**: 「〇〇定期」「これが義務教育の敗北か」（無知への煽り）「ソースは信長の野望」（ゲーム知識への自虐）「〇〇ニキ」（詳しい人）
  - **辛口短評**: 打線の順位に「3番ゴミ過ぎて草」「タレント多すぎやろ」「頭いかれすぎやろ」と一言ツッコミ
  - **打線フォーマット**: 「1 中 〇〇／2 左 〇〇／…／9 投 〇〇／抑え 〇〇 中継ぎ 〇〇」。イッチが打順発表→住民が「〇番は△△の方が」と入れ替え議論
- 2026-06-16: コーパスに大塩平八郎まとめ3本追加（腹筋崩壊/哲学ニュースnwk/歴史的速報・計147レス）。**「しょぼい失敗を皮肉る＋賛否が割れる人物」系**の型を抽出👇
  - **箇条書き論破フォーマット**: 失敗を「・〜のなさ」で畳みかける。例「・決起直前に内通者を複数出す人望のなさ／・救民を掲げて大砲ぶっぱなし火災(大塩焼け)を広げただけの計画性のなさ／・わずか半日で鎮圧される規模のショボさ」。1レスで複数の欠点を列挙して落とす。創作スレの"知識ニキの逆バージョン（こき下ろしニキ）"に使える
  - **「〇〇定期」オチ**: 既出ネタを一言で締める「半日で鎮圧定期」。短文死にレス風に効く
  - **賛否割れの摩擦**: 同一人物で住民が二分する。煽り側「塩カス無能すぎて草」「大勢に迷惑かけたテロやろこれ」「嫉妬民湧きすぎて草」 vs 擁護側「乱起こす時私財を寄付したんだよな、不覚にも泣けた」「当時やれるベストは尽くした」。光/影を1スレに同居させる時の見本
  - **なりきり寸劇レス**: キャラのセリフを妄想で書く「平八「各地の陽明学者に檄文送ったわ。これで諸国の同志が呼応して立ち上がるやろなあ」」→現実は誰も来ない、のオチ。脱線パートに1個挟むとリアル
  - **教科書/語呂ツッコミ**: 「なんで教科書にのっとるんやろ」「語呂が良くて印象に残りやすいだけやろ」「元役人が反乱起こすのがヤバくて名前が売れたんやろ」＝有名な理由をメタに茶化す
  - **時代の暇さ煽り**: 江戸の平和を「1700年代はイベントなさ過ぎやろ」「停滞の時代やろ」と退屈扱い→だからこの乱が目立つ、の文脈作りに使える
- 2026-06-16: コーパスに今川義元・桶狭間2本＋ガバガバ歴史速報の日本史5本（平安鎌倉/賤ヶ岳IF/竹取物語/信長家臣/上杉謙信・計601レス）追加。新しい型👇
  - **IF・たられば議論型**: 「義元が勝っとったら勝家とか秀吉はどうなっとったんやろか」→「勝家はそのまま今川に鞍替えやろな」。"もし○○だったら"を起点に住民が分岐を妄想する。賤ヶ岳IFも同型。創作スレの後半（脱線→盛り上がり）に強い。突っ込み役の「○○でたらればってナニかあるんか？」も定番
  - **「○○で詰む定期」**: 勢力を持ち上げた直後に冷や水「三好で詰む定期」。過大評価を一言で折る短レス
  - **雰囲気・情緒語り型**: 「平安～鎌倉の雰囲気なんか好き」系。煽りより共感ベースで進む。「これはわかる」「明治～戦前も好き」と同意が連なり、合間に生活妄想「みんな歯が黄色くて口臭いんやろな」「子供は普段どんな事話してたんやろ」。光寄り・しみじみトーンの参考
  - **タイムスリップ仮定**: 「北条政子に2018年の化粧品渡したらその時代のno.2になれそう」。現代の物/知識を持ち込む妄想は鉄板の脱線ネタ
  - **なりきり大喜利型**: 「信長「お前これやれ」家臣「はい…(こんなん絶対無理やん)」」スレは住民が家臣になりきって無茶振りオチを大喜利する。1スレまるごとロールプレイ。創作スレで1パート寸劇化する時の手本
  - **ゲーム知識ベースの強さ議論**: 「信長の野望って何で上杉謙信大正義なんや？」。"ソースは信長の野望/センゴク(漫画)"と出典が娯楽作品なのを自覚しつつ強さを語る。史実と娯楽がごちゃ混ぜなのがリアル
- 2026-06-16: ガバガバ歴史速報「なんJ日本史部」から日本史12本（信長家臣/大谷吉継/今川/真田/秀吉/武田/徳川15代/鑑真/大河島津/教科書名言/恋の○○・計1574レス）を追加学習。新しい型👇
  - **お題大喜利型（言葉遊び）**: スレ全体が1つのお題への回答連投。「日本史用語に"恋の"を付けるとラブソング」→「恋の本能寺の変」「恋の御成敗式目」「恋の十七条憲法」が延々続く。住民が短い回答を投下し合うだけで伸びる。創作スレの軽い箸休めパートに使える
  - **大河ドラマ・メタ議論型**: 「なんで○○は大河化されない？」→「視聴率やろ」「主人公が一般に浸透してないと取れん」「○○でええやろ」と代替候補を出し合う。地元自慢・観光誘致ネタ（「柳川で立花宗茂の大河誘致してた」）も混ざる。創作の現代視点パートに
  - **教科書名言・3大○○型**: 「日本史教科書3大名言『どうだ明るくなつたろう』『堂々退場す』」。"3大○○"とお題を振り→住民が候補と4つ目を出し合う定番フォーマット
  - **人気と実力を切り分ける論法**: 人物評価スレで「実績だけ見れば○○より上やろ」「人気は脚色のおかげ、忠臣蔵みたいなもん」「真田丸で前田を抑えたのは史実やろ」と、創作補正と史実を腑分けして議論する。過大/過小評価ネタの厚みが出る
  - ⚠️ コーパス中に伏字スラング（ジ●ップ等）が混じる箇所あり。**学習素材として読むだけ**で、生成時は[[台本執筆ルール]]通り伏字・差別語を一切使わない
- 2026-06-16: 本文2500字以上の長文スレ18本を厳選学習（今川義元18千字/つまらん時代/家康江戸首都/榎本武揚/秀吉IF・計2804レス13万字）。**「厚い議論スレ」の作り方**＝以下を厚めに盛ると長く読ませられる👇
  - **通説否定・デマ仕分け構文（知識ニキの主力）**: 「○○ってよく言われるけど実際は〜」「家康の脱糞は史料あるけど"石川に食わせた"は完全に2ch発のデマ」「誤解しとる奴多いけど〜やで」。"通説→실은違う"の腑分けが議論の厚みの核。創作スレの転換点で長文ニキに1回やらせる
  - **史料・出典で権威付け**: 「吉川弘文館の人物叢書」「歴史文化ライブラリー」等の実在シリーズ名や一次史料を出して説得力を盛る。ただし創作スレでは存在しない史料の捏造はNG（ファクトチェック）。実在の有名史料名に留める
  - **比較で格付け**: 「日ノ本の大半を収めたノッブと東海止まりの弓取りを比較したらあかんやろ」。2人を並べて優劣を断じる→反論が湧いて伸びる
  - **判官びいき擁護**: 「○○が一番可哀想」「定期的にディスられる朝倉家に憤怒」「小早川秀秋、能力が糞雑魚に設定されてて可哀想」。負け組・過小評価キャラを庇うレスが摩擦を生む
  - **長文ニキと短レスの落差を意図的に作る**: 厚い議論スレほど、3〜5行の解説ニキの直後に「長い」「3行で」「で、結局どっちが強いん？」の即レスを挟むと本物感が増す（緩急パターンの実例）
  - ⚠️ 長文スレは禁止語（殺し屋/殺される/死等）が地の文に混じりがち。生成時は必ず言い換え
