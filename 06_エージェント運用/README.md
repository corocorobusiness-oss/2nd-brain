# 06_エージェント運用

Obsidian x Claude / Codex を24時間に近い形で動かすための管制塔。

ここは作業成果物の置き場ではなく、AIが「今の状態」「未処理」「確認待ち」「健康状態」「判断ログ」を見るための運用レイヤー。

## 使い方

1. 新しいAIエージェントは `00_システム/10_Agent/` を読んだ後、`00_司令塔/NOW.md` を確認する
2. 雑な依頼や未分類メモは `10_Inbox/未処理.md` に入れる
3. 人間判断が必要なものは `10_Inbox/要確認.md` に置く
4. 実行する作業は `20_実行キュー/` で状態管理する
5. 自動化やWatchtowerの要約結果は `30_ヘルスチェック/` に残す
6. 重要な決定は `40_判断ログ/決定事項.md` に残す

## 保存しないもの

- トークン
- Cookie
- セッション
- 認証情報
- 生ログ
- キャッシュ
- 外部サービスの秘密情報

生データではなく、判断に使える要約だけを残す。
