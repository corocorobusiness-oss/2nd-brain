# 明鏡コミュニティ「開発室」Q&A・トラブルシュート集

> AI&マーケ学習コミュニティ「明鏡」🎮開発室の実ログから抽出（2026-06-08収集・154件）。
> Roblox教材（お寿司職人）を進める仲間のリアルな詰まりポイントと解決法。制作で詰まったらまずここ。

## 🔧 環境構築（Day1）のトラブル
- **MCP設定の入口が移動した**: Roblox Studio側の「Enable Studio as MCP server」を探すのに時間がかかる事例。場所が変わっている可能性 → ChatGPTにスクショ見せながら探すと見つかる
- **Claude接続 failed の原因候補**: ①mcp.batの文字化け ②**日本語ユーザー名の影響**（Macのユーザー名が日本語だと不具合の可能性）
- **「Enable Studio as MCP server」が緑にならない**事例あり
- **Codex / Antigravity でも連携できる**（Claude以外でもOK。ChatGPT課金分=Codexで代替可能）
- 教材通りに進めれば詰まらずMCP連携できた、という報告も多数

## 🗺️ マップ作成（Day2）
- **カメラ視点が正面からしか見えない** → **画面右上の「サイコロみたいなアイコン」**をクリックで角度変更（ViewSelector）。カメラ操作は教材GPTs「ゲーム完成ナビ」に聞くと答えてくれる
- 座標手打ちは2時間かかるが、AI出力に任せれば一瞬 → 「手作業の後にAIで一瞬」の体験が好評
- マップ完成で一気にゲーム感が出る

## 🪙 モデル作成（Day3）— 最多の詰まりポイント
- **コインのテクスチャが反映されない** → 教材内で配布されたアセットIDで進める方が安心
- **だんご屋（ショップ）のリメッシュで崩れる** → **装飾を減らしてシンプルにする**と成功（店内の装飾もなくすとうまくいく）
- **Meshyで看板の日本語が崩れる** → Meshyは日本語再現が苦手。「**3Dモデルは無地の看板にして、後から日本語を画像として貼る**」のが定石
- コイン配置・モデル作成が分からなければ**AIに丸投げ**でOK（みんなやってる）

## 💾 Roblox本体のトラブル（重要）
- **アカウント停止（BAN）が起きる**: 作業中に「停止」を食らう人が複数。1回目7日間、異議申し立てで約12時間後に解除。2回目は6時間後。深夜時間帯に起きやすい → **異議申し立て＋届いたメールに返信で対応**
- **停止を食らうとStudioの保存も機能しなくなる** → こまめ保存しても消えることが。運が良ければ残っている
- **保存できなかった時** → Claudeに「さっきまでやったところを復元して」と頼んで復旧した事例（2-3回成功）
- **サムネイルのページが真っ黒になり設定できない**事例あり
