# Roblox事業 設計図（全体像 v1 2026-06-09）

## 🎯 コンセプト（一言）
「AIだけでゲームを作る人」として発信し、教材アフィリ＋受託＋自作ゲーム収益の3エンジンを回す。

## 🧱 4つの構成要素と関係
```
        ┌─────────────┐
        │  ① 制作      │ ← Claude+MCP / 教材14日フロー
        │ ゲームを作る  │   成果物: 公開ゲーム＝ポートフォリオ
        └──────┬──────┘
               │ 作る過程・つまづき・完成品が素材に
               ▼
        ┌─────────────┐
        │  ② 発信      │ ← X(毎日) / Substack(週1) / note(週1)
        │ 過程を見せる  │   役割: 集客＝信用構築＝営業
        └──────┬──────┘
               │ 発信が見込み客を連れてくる
       ┌───────┼───────┐
       ▼               ▼
┌────────────┐  ┌────────────┐
│ ③ アフィリ   │  │ ④ 受託      │
│ 教材を紹介   │  │ ゲーム作る   │
│ 50%還元・即  │  │ 高単価・実績後│
└────────────┘  └────────────┘
       ＋自作ゲームのゲーム内収益（パス/広告）
```

## 💰 収益エンジン（立ち上がり順）
| # | エンジン | 入金まで | 単価 | 立ち上げ条件 |
|---|---|---|---|---|
| 1 | **アフィリ** | 即 | ¥2,450〜4,900/件 | 発信＋紹介リンクだけ |
| 2 | **ゲーム内収益** | 中 | 小〜中 | プレイ数が必要 |
| 3 | **受託** | 後 | 高（数万〜） | 自作実績＋発信の信用 |

