# 0から始めるBUYMAロードマップ 全教材読了台帳・統合構成案

作成日: 2026-07-12

## 結論

既存の12章構成を土台にしつつ、初心者が実践する順番を崩さず、公開教材・子ページ・PDF・管理表の内容を1本へ統合する。

ただし、旧教材の文章をそのまま連結はしない。現在のBUYMA規約、外注先サービスのルール、フリーランス法、個人情報・画像権利に合わない箇所は、次のいずれかに変換する。

1. 現在の正しい手順へ更新する
2. 「やってはいけない旧手順」として注意喚起に使う
3. 数値・報酬・利益率は固定ルールではなく、当時例または計算例にする
4. 法的な文面は完成テンプレートとして断定せず、確認項目と専門家確認へ変える
5. 各節を「原本移植」「公式情報による修正」「編集上の並べ替え」に区分する

原本と公式情報に根拠がない新規の実践手順は、本構成へ追加しない。必要な場合は「要ユーザー承認」として本編から分離する。

## 公開前に必ず直す旧記述

### 1. アカウント・外注化

- 「複数アカウントで検証」は削除する
- 外注候補に新規BUYMAアカウントを作成・出品・退会させる「パッチテスト」は実施しない
- 代替の選考手順は本構成で新規創作せず、利用サービスとBUYMAの現行規約を確認後、ユーザー承認を得て別途設計する
- 外注パートナーの行為を含め、責任は登録者に帰属することを明記する

公式確認:

- https://qa.buyma.com/shopper/reg/13732.html

### 2. 自動出品・大量出品・外部プログラム

- 原本にある自動出品ツール、月5,000品、大量出品の具体手順は推奨手順として収録しない
- BUYMAが許諾していない外部プログラム、サーバーへ過度な負荷をかける自動出品、スクレイピングを使わない
- 特定ツール名を「現在利用可能」として掲載しない
- 原本から残すのは、継続的に出品・改善する運営思想と、手動運用の役割分担だけにする

公式確認:

- https://qa.buyma.com/shopper/tips/30118.html

### 3. 直営店から購入者への直送

- 買付先店舗スタッフによる検品だけで購入者へ直送する手順は掲載しない
- 店舗スタッフはBUYMA上の買付パートナーとして認められない
- 出品者本人または共に買付を行うパートナーが検品する流れへ変更する
- 章名も誤解を避けて「直営店連携仕入れ」とする

公式確認:

- https://qa.buyma.com/shopper/order-to-delivery/4205.html
- https://qa.buyma.com/shopper/order-to-delivery/4201.html

### 4. 商品画像・モデリング

- ECサイト画像を保存・加工して使う説明は削除する
- 加工しても、権利者の許諾がなければ使用できない
- 他ショッパーの画像・文章・商品構成をコピーする「モデリング」は、需要理由と運営構造の分析へ置き換える
- 画像検索は仕入先調査の補助に限定し、画像の転載は禁止利用を認める説明にしない

公式確認:

- https://qa.buyma.com/shopper/prohibited-item/15733.html

- 「日本未入荷」「限定」「大人気」「残り1点」「著名人愛用」は、根拠を確認できる場合だけ使う

### 7. 募集・契約・個人情報

- 「規制を別表現ですり抜ける」は削除する
- クラウドワークスでは、無在庫販売、出品作業・商品登録・在庫管理代行、代理購入、アカウント共有を必要とする依頼が禁止されているため、原本のBUYMA外注募集先として使わない
- 契約前面談・外部連絡先交換・直接取引にも各サービスの制限があるため、サービス別の「委託可能・禁止業務表」を作る
- ランサーズ、ココナラなど他サービスも、現行ガイドラインを個別確認するまで利用可能と断定しない
- 募集文は業務、場所、報酬、発注者情報を正確・最新にする
- 「女性限定」「主婦限定」、年齢・家族構成・身分証コピーなど、業務に不要な条件や個人情報を安易に集めない
- 報酬額、支払期日、検査日、業務内容を文書または電磁的方法で明示する
- 旧業務委託契約書はそのまま使える完成版として掲載しない
