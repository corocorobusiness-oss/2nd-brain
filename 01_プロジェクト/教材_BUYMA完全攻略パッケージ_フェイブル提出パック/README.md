# フェイブル5提出パック

このフォルダは、Brain教材「BUYMA完全攻略パッケージ」をフェイブル5に第三者レビューしてもらうための提出パックです。

## 使い方

まず、以下をフェイブル5に貼る。

- `00_フェイブル5への依頼文.md`

その後、可能なら以下の順番で対象ファイルを添付または貼り付ける。

1. `../教材_BUYMA完全攻略パッケージ_商品設計.md`
2. `../教材_BUYMA完全攻略パッケージ_Brain販売ページ下書き.md`
3. `../教材_BUYMA完全攻略パッケージ_Brain本文統合目次案.md`
4. `../教材_BUYMA完全攻略パッケージ_Brain入稿本文.md`
5. `../教材_BUYMA完全攻略パッケージ_販売前チェックリスト.md`
6. `../教材_BUYMA完全攻略パッケージ_素材リンク台帳.md`

`素材リンク台帳` は購入者に見せるための資料ではなく、Notion教材本文をBrainへ移植するための管理用です。フェイブル5へ渡す場合も「外部公開しない前提の移植元一覧」として扱う。

## フェイブル5に必ず見てもらうこと

- Brain商品として「1つの教材」に見えるか
- Notionリンク集ではなく、Brain本文内完結にする方針が守れているか
- 無料部分と有料部分の境界が自然か
- 9,800円から販売開始する設計が妥当か
- BUYMA、ブランド、ショップ情報、収益表現まわりで危ない表現がないか
- Notion本文を移植する前に、追加すべき章や注意書きがないか

## レビュー結果の保存先

フェイブル5から返ってきた内容は、以下のファイル名で保存する。

- `../教材_BUYMA完全攻略パッケージ_フェイブル5レビュー結果_2026-07-09.md`

レビュー指摘は、次の3つに分けて処理する。

1. 販売前に必ず直す
2. 本文移植後に再確認する
3. 発売後の追記でよい
