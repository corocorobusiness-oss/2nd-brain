# フェイブル提出パック

このフォルダは、Brain教材「7日で作る Codex × Obsidian 自分専用AI秘書」をフェイブルに第三者レビューしてもらうための提出パックです。

## 最初に渡すファイル

まず、以下をフェイブルに貼る。

- `00_フェイブルへの依頼文.md`

## 添付するファイル

可能なら、この提出パック全体を添付する。

添付できない場合は、以下の順で貼る。

1. `01_設計と販売ページ/商品設計.md`
2. `01_設計と販売ページ/Brain販売ページ完成版.md`
3. `02_本文/Day0本文.md` 〜 `02_本文/Day7本文.md`
4. `03_特典テンプレ/`
5. `01_設計と販売ページ/画像挿入リスト.md`
6. `04_画像/`

## 必ず聞くこと

フェイブルには、以下の3点を必ず聞く。

1. この `2nd-brain` ファイル構成は初心者にとって最適か
2. 教材を読めば実際に使える状態まで作れるか
3. 980円〜2,980円で販売してよい品質か

## 返ってきたレビューの扱い

レビュー結果は、そのまま鵜呑みにしない。

次の3つに分ける。

- 販売前に必ず直す
- できれば直す
- 発売後の追記でよい

重大な修正だけ販売前に反映し、細かい改善は発売後の追記でもよい。
