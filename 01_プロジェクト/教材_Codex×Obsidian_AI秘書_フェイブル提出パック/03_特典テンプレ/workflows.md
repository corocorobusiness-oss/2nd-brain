# Workflows

## メモ整理

目的:
- デイリーノートや `03_memos/` にあるメモを整理する

手順:
1. メモを読む
2. タスク、アイデア、学び、後で確認に分ける
3. 今日やることを3つまでに絞る
4. 確認待ちを分ける
5. 明日の一手を1つ出す

出力:
- 今日やること
- タスク
- アイデア
- 学び
- 確認待ち
- 明日の一手

## タスク整理

目的:
- 今の目標に合うタスクを優先する

手順:
1. `goals.md` を読む
2. 今日のデイリーノートを読む
3. タスクを抜き出す
4. 今日やること、確認待ち、後でに分ける
5. 今日やることは3つまでにする

出力:
- 今日やること
- 確認待ち
- 後で
- 最初にやること

## 企画出し

目的:
- メモと目標から、発信やコンテンツ販売につながる企画を作る

手順:
1. `profile.md` を読む
2. `goals.md` を読む
3. `03_memos/` を読む
4. 出口を決める
5. 企画案を5つまで出す
6. 一番おすすめの案を1つ選ぶ

出力:
- 企画タイトル
- 誰向けか
- 読者の悩み
- 伝える内容
- 今の目標に合う理由
- 最初に作るべき一手

## 週次メンテ

目的:
- `2nd-brain` の中身を軽く見直す
- ルールや手順を育てる

手順:
1. 今週のデイリーノートを確認する
2. 完了したタスクを振り返る
3. 残っているタスクを整理する
4. よく繰り返した作業を探す
5. ルールに追加すべき失敗や注意点を探す
6. 来週の最初の一手を決める

出力:
- 今週できたこと
- 来週やること
- `rules.md` に追記した方がいいこと
- `workflows.md` に追記した方がいいこと
- 不要になったメモやタスク
