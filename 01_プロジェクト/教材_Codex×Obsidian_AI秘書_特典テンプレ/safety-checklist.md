# AIに任せていいこと・任せないことチェックリスト

> 置き場所: 自由（`00_system/` に置いておくのがおすすめ）。AIに何かを任せるか迷ったときに見返す用です。

## 任せていいこと

- [ ] メモの分類
- [ ] タスクの抜き出し
- [ ] 今日やることの整理
- [ ] 企画案の作成
- [ ] 文章の下書き
- [ ] 日報の作成
- [ ] 週次振り返りの作成
- [ ] 関連メモの探索

## 確認してから進めること

- [ ] 既存ファイルの編集
- [ ] `goals.md` の変更
- [ ] `rules.md` の変更
- [ ] `workflows.md` の変更
- [ ] 販売ページや投稿文の完成版作成
- [ ] 複数ファイルをまたぐ整理
- [ ] 重要な方針変更
- [ ] 価格や販売条件の変更

## 任せないこと

- [ ] ファイル削除
- [ ] 外部投稿
- [ ] 課金や購入
- [ ] パスワードの保存
- [ ] APIキーやトークンの保存
- [ ] 個人情報の保存
- [ ] お金に関わる最終判断
- [ ] 自分の代わりに重要な決断をすること

## AIに貼らないもの

- [ ] パスワード
- [ ] APIキー
- [ ] 認証トークン
- [ ] Cookie
- [ ] クレジットカード番号
- [ ] 銀行口座情報
- [ ] 住所や電話番号
- [ ] 顧客情報
- [ ] 外に出せない売上や契約の詳細

## 外に出す前の確認

- [ ] 数字や実績に間違いがない
- [ ] できない約束をしていない
- [ ] 誇張した表現になっていない
- [ ] 読者に誤解される表現がない
- [ ] 個人情報や秘密情報が入っていない
- [ ] 最後は自分で投稿・販売開始する
