## 3. 公開範囲

- [ ] Notionページが購入者だけ見られる設定になっている
- [ ] 編集権限を渡していない
- [ ] 複製可否を意図通りに設定した
- [ ] 外部に見せたくない個人メモが混ざっていない
- [ ] 他案件や個人情報へのリンクが混ざっていない

## 4. 表現チェック

- [ ] 「必ず稼げる」を使っていない
- [ ] 「誰でも月収○万円」を使っていない
- [ ] 「放置で収益化」を使っていない
- [ ] 「完全自動で稼げる」を使っていない
- [ ] 「再現性100%」を使っていない
- [ ] BUYMA公式教材と誤認される表現がない
- [ ] 成果保証ではなく、実践ベースの教材として説明している

## 5. 権利・規約まわり

- [ ] 自分で作った教材だけで構成している
- [ ] 他人の教材本文を流用していない
- [ ] 他人の有料コンテンツを転載していない
- [ ] 画像やスクショに権利的に危ないものがない
- [ ] ブランド名やショップ情報の出し方を確認した
- [ ] BUYMAやショップの規約違反を推奨する内容になっていない

## 6. Brain販売ページ

- [ ] タイトルを決めた
- [ ] サブタイトルを決めた
