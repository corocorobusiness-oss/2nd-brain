# 平安時代雑学 勝ち型メモ

作成日: 2026-07-06

## 結論

次の 7/10(金) 枠「平安時代雑学」は、単なる豆知識羅列ではなく **偏見レス → 知識レスで反転 → 生活/恋愛/権力/医療/怨霊へ展開** の型で作る。

主軸はこれ。

> 平安時代、雅どころか想像以上にカオスだった

安倍晴明の次なので、ラストだけ陰陽師・怨霊・祈祷へ接続する。全編を陰陽師に寄せすぎない。

## 学習した型

### 0. ナレーションの言い回しは既存スキル準拠

雑学モードでも、通常動画の雰囲気は変えない。  
冒頭・前置き・解説・締めのナレーションは `youtube-script-parts` の既存ガイドの言い回しを優先する。

- ナレでは「スレ民たち」ではなく、「2ちゃんねるの歴史オタクたち」「議論」「疑問を持つ者」など既存スキルの表現に寄せる
- 前置きの締めは原則「引き続き、2ちゃんねるの歴史オタクたちの議論を見ていこう。」系を使う
- 雑学モード専用の砕けたナレ口調を新設しない
- 変えるのは構成だけで、チャンネルの通常トーンは変えない

### 0-2. 雑学モードのファイル分担

リサーチ段階では、タイトルと売り方を決める。  
ただし、ここで決めるタイトルはあくまで仮タイトル扱いにする。

- `brief.json`: 雑学モード宣言、仮タイトル、タイトル候補、タイトルの約束、売り方、使えそうな素材候補まで
- `zatsugaku_plan.md`: 前半/後半の役割、後半の回収順、雑学モード尺ゲート、AI臭防止ルール、軽ゲート結果
- 創作スレ生成: レスの流れ、茶々、勘違い、脱線、自然な会話の肉付け
- `manifest`: いつも通り。雑学構成やAI臭ルールの置き場にはしない

`brief.json` には、レス順、具体的なセリフ、前置き/解説/締め文案、後半の細かい回収順、manifest的な組み立て指示は入れない。

雑学モードだけ、manifest には `content_mode: "zatsugaku"` と `length_policy.profile: "zatsugaku"` を入れる。  
これは構成メモではなく、組み立て時の機械ゲートを発火させるためのフラグである。通常のオリジナルスレには入れない。

### 0-3. タイトル確定ゲート

リサーチ段階のタイトルは `working_title` とし、`title_provisional: true` の扱いにする。

`zatsugaku_plan.md` 完成後に「この中身はタイトルに負けていないか」を確認し、創作スレ生成前に初めて `final_title` としてロックする。

平安時代雑学の仮タイトル:

> 【2ch歴史】歴史オタクの雑学で平安時代が面白くなりすぎたwww【ゆっくり解説】

タイトルの約束:

> 平安時代のイメージが、雅からカオスに変わる

### 0-3.5. 雑学モード尺ゲート

大カテゴリ雑学は、短レスの雰囲気だけ作ると動画尺と満足感が足りなくなる。  
そのため、雑学モードだけ以下をハードゲートにする。

- スレ①: 最低2,500字
- スレ②: 最低2,500字
- スレ本文合計: 最低5,000字

これはスレ本文のみで判定する。  
冒頭、前置き、解説、締めで不足分を埋めるのは不可。

`creative-thread-gen/qa_check.py` の `本文字数: 2500 PASS` は、単体スレとしての最低QAであり、雑学動画としての完成尺ではない。  
雑学回では、前半/後半の両方が2,500字を超えるまで本ゲートPASSにしない。

### 0-4. 前半/後半の役割

雑学回は、前半と後半で温度差を作る。

- 前半: 入りやすい雑学。笑える、変、意外、教科書イメージと違う
- 後半: 深い、怖い、重い話で回収する

平安時代雑学では、前半は「雅そうに見えて生活が変すぎる」、後半は「治安・病気・祈祷・怨霊で普通に怖い」とする。

前半レスは綺麗に締めない。  
裏の印象として「平安人、雅というよりクセが強すぎる」を残し、前置きパートで後半へ空気を切り替える。

### 0-5. 後半の回収順

後半の回収順は `zatsugaku_plan.md` にだけ置く。  
`brief.json` と `manifest` には入れない。

平安時代雑学の裏設計:

> 治安 → 病気/死生観 → 祈祷依存 → 怨霊観 → 陰陽師で回収

ただし台本上では分類名を出さない。  
「次は治安」「次は病気」のように見出し化せず、質問と反応で自然に重くする。

祈祷・怨霊・陰陽師は連続させすぎない。途中に茶化し、勘違い、日常ネタを混ぜる。

### 0-6. 前置き/解説パート

前置きパートは雑学専用の型を作らない。  
`youtube-script-parts` の前置き型をそのまま使い、役割は前半の軽い雑学から後半の少し重い雑学への空気転換だけにする。

平安時代雑学の前置き方向:

```text
ここまでは、
平安時代の貴族文化や暮らしについての
さまざまな議論を見てきたが、
「でも実際の平安京って、
普通に暮らしやすかったの？」
「夜の都や病気はどうしていたの？」
と、当時の暮らしに疑問を持つ者も現れた。
たしかに、華やかな文化の一方で、
人々の生活にはどのような不安があったのだろうか。
引き続き、2ちゃんねるの歴史オタクたちの
議論を見ていこう。
```

解説パートは通常運用。  
最低2本、前半1本・後半1本を基本にし、雑学ごとに細かく入れすぎない。スレ本文の復唱ではなく、理解に必要な背景だけを補う。

### 0-7. AI臭防止ルール

基本骨格は裏に持つ。

```text
偏見レス
茶々・ツッコミ
知識レスで反転
リアクション
次の雑学
```

ただし、これを綺麗に完走させるのは半分から6割まで。連続2回以上、綺麗に完走させない。

- いきなり知識レスから入る
- ツッコミだけで終わる小ネタを入れる
- 勘違いレスを混ぜる
- 未回収の疑問を少し残す
- 日常ネタに脱線する
- 短い `>>` 返信を混ぜる
- 話題を急に曲げる
- 長文3連続は禁止
- 同じ語尾3連続は禁止
- 役割固定は禁止
- 毎回「正解発表」にしない
- 「諸説」「地域差」「時代差」「全部がそうではない」を混ぜる

ただし崩しすぎない。軸・切替・余韻は残す。  
最終チェックは「構成表を見なくても、読んで型が見えたらAI臭あり」。

### 0-8. レビューゲート

雑学モードのレビューは2段にする。

軽ゲート: `zatsugaku_plan.md` 後。5分チェック。

- 雑学回として入口が広いか
- 前半/後半の役割が分かれているか
- 雑学モード尺ゲートが定義されているか
- 通常の2ch歴史トーンからズレてないか
- AIっぽい固定ループになってないか
- 史実・規約で危なそうな話題がないか

本ゲート: 創作スレ生成後、`manifest` 前。10〜15分上限。

- タイトルの約束に中身が負けてないか
- 冒頭1〜2分で「この動画は当たりそう」と思えるか
- 前半で満足せず、後半を見る理由が残っているか
- スレ①2,500字以上、スレ②2,500字以上、スレ合計5,000字以上を満たしているか
- レスとして自然か。雑学解説ではなく会話に見えるか
- 3ネタ連続で同じ話運びになってないか
- レスが便利に説明を進めすぎてないか
- 解説が台本を支配してないか
- 危険表現・大きな史実ミスがないか
- 平安では、安倍晴明を万能役にしすぎてないか

判定は `PASS / MINOR / 差し戻し`。

- `PASS`: 尺ゲートを含めて満たした場合だけmanifestへ
- `MINOR`: 小修正して通す
- `差し戻し`: タイトルと本文が別物、雑学羅列、AI定型ループが全体に出ている、尺不足、危険表現、主要史実ミスなど

### 1. 偏見スタート

本物スレは、最初から知識を並べない。最初は雑でいい。

- 「平安時代ってろくな時代じゃなかったよな」
- 「貴族以外は人権なさそう」
- 「庶民に生まれたら地獄、貴族に生まれても糖尿病」
- 「寒いし風呂もないし」

ここで視聴者の先入観を代弁する。

### 2. すぐ反転させる

偏見だけで進めると暗い。早めに知識レスで反転させる。

- 死刑がなかった/検非違使が現場判断していた、など制度のズレ
- 平安は地味ではなく、遺物が劣化して見えるだけで本来は極彩色
- 貴族文化は華やかだが、都の外や庶民生活は別世界
- 女性の地位や婚姻は、単純な男尊女卑では片付かない

### 3. 雑学は「生活」から入れる

最初から政治史に入らない。登録者向けの入口は生活感が強い方がよい。

推奨順:

1. 食事・歯・病気
2. 家・寒さ・服装
3. 恋愛・結婚・女性
4. 貴族と庶民の格差
5. 検非違使・治安
6. 祈祷・陰陽師・怨霊
7. 文化・文学で少し回収

### 4. 1ネタは短く、たまに長文知識レス

雑学系は短レス連打だけだと軽すぎる。  
ただし長文解説だらけにすると維持率が落ちる。

目安:

- 短レス 3〜5本
- ツッコミ 1本
- 知識レス 1本
- 「知らんかった」系リアクション 1〜2本

この小ブロックを 6〜7個つなぐ。

### 5. 後半に「怖さ」を入れる

平安の強みは、雅だけで終わらず不気味さへ落とせるところ。

後半候補:

- 病気になると祈祷に頼るしかない
- 怨霊や物の怪が政治判断に影響する
- 貴族社会は華やかだが、後ろ盾を失うと一気に詰む
- 都の中心と端で暮らしが違いすぎる
- 右京の荒廃、治安、検非違使

安倍晴明回の次なので、ここで関連流入を拾う。

## タイトル方針

本線:

> 【2ch歴史】歴史オタクの雑学で平安時代が面白くなりすぎたwww【ゆっくり解説】

サブ案:

- 【2ch歴史】平安時代、雅どころかカオスすぎた件www【ゆっくり解説】
- 【2ch歴史】平安時代って実はろくでもない時代だったんじゃね？【ゆっくり解説】

今回は大カテゴリ雑学のテストなので、タイトルは本線の「歴史オタクの雑学で平安時代が面白くなりすぎたwww」を優先する。

## やらないこと

- 安倍晴明の続編にしすぎない
- 怨霊・陰陽師だけで一本作らない
- 雑学を20個ただ並べない
- 政治史から入らない
- 史実の怪しい俗説を断定しない
- 「実は〜という真実」型に逃げない
