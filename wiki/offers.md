# Offers

商品、サービス、提案、価格、提供価値を整理するページ。

## 管理するもの

- 商品・サービス名
- 誰のどんな問題を解決するか
- 価格
- 提供範囲
- 納品物
- 申し込み導線
- よくある質問
- 改善案

## 現在のオファー候補

| 種類 | 内容 | 状態 |
|---|---|---|
| YouTube収益 | 2ch歴史系チャンネルの広告収益 | 既存の主柱 |
| Brain教材 | AI/Roblox/Codex/Obsidianなどの教材化 | 候補・一部素材あり |
| Note記事 | AI活用、制作フロー、事業運用の文章商品 | 候補 |
| メルマガ | 教材やNoteへの導線 | 候補 |
| X長文ポスト | 認知・信頼形成・販売導線 | 候補 |
| Robloxゲーム内収益 | ゲームパス、広告、体験課金 | 検証前 |
| Roblox受託制作 | AIでRobloxゲーム制作を請ける | 実績構築後 |

## AIの使い方

- 商品説明を改善する
- 顧客の悩みからオファーを作る
- 価格や提供範囲の矛盾を見つける
- 発信やLPの材料に変換する
- YouTubeやRobloxの制作ログを商品化候補に変換する
