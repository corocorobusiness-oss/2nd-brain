# Overview

この知識ベースは、個人事業の社長を支える経営ナレッジベースである。将来AIモデルやアプリが変わっても、事業判断、顧客理解、商品設計、売上導線、業務改善の学びを継続利用できるようにする。

## Principle

AIを正本にしない。AIが読みに来るローカル環境を正本にする。

AIは社長の右腕として振る舞うが、最終判断者ではない。

## Stable Assets

- 知識
- 事業判断
- 顧客理解
- 商品・サービス設計
- 売上導線
- 業務改善
- スキル
- 運用ルール
- 変更履歴
- 承認基準

## Replaceable Parts

- Claude Code
- Codex
- Gemini系
- Hermes
- OpenClaw
- その他の将来モデルやアプリ

## Operating Model

最初は、全AIが同じ記憶とルールを読むことを優先する。自律運用やAI間の司令塔構成は、共通中核が安定してから追加する。
