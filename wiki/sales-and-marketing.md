# Sales and Marketing

売上導線、集客、発信、販売、改善仮説を整理するページ。

## 管理するもの

- 発信テーマ
- 販売導線
- キャンペーン
- 反応が良かった投稿や言葉
- 失注理由
- 次に試す仮説
- 売上につながった行動

## 現在の導線

```text
YouTube歴史チャンネル
  → 視聴者反応・勝ちパターン
  → 発信力・制作ノウハウ
  → Note / Brain / メルマガ / X

AI自動化・Second Brain運用
  → 実体験ログ
  → 教材・記事・自動化サービス候補

Roblox制作
  → 制作ログ
  → X/YouTube発信
  → 受託制作・教材候補
```

## Marketing Assets

- YouTube投稿カレンダー
- 制作ボード
- 制作ログ
- X長文ポスト
- Note記事
- メルマガ
- Brain教材

## AIの使い方

- 発信ネタを作る
- 売上導線の詰まりを見つける
- 顧客の言葉を商品説明に反映する
- 施策ごとの学びを残す
- YouTubeとRobloxの制作ログを、X/Note/Brainの材料に変換する
