# 冒頭パート作成ガイド

## 目的
ユーザーから提供された「テーマ」や「対象となる武将・事象」などをもとに、YouTube台本の「冒頭パート」を**ゼロから書き起こす**。過去に作成された台本と**完全に同一のフォーマット・トーン**で新規出力すること。

## 出力ルール・制約

AIは、以下の厳密なルールに従って**ゼロから文章を生成**し、過去の原稿の王道パターン（フック、問題提起、企画紹介、メリット）と見分けがつかないレベルで完全にトーンとフォーマットを統一すること。

### 1. 文字数と最重要目的
- **最大文字数とセル分割**：**200〜350文字**で作成し、切りのいいところで**2～3行ずつ**セルを分けて（ブロック化して）ください。
- **最重要目的**：冒頭文作成で最も意識してほしい点は**『本編への興味を湧かせること』**です。冒頭文で『本編を見たい！』と思わせる意識を忘れないでください。

### 2. 必須要素
必ず以下の2つの要素を文章中に加えること。
- **視聴者への問いかけ**：「果たして、〜一体どうすればよかったのだろうか？」など、視聴者の興味を惹きつける疑問を投げかける。
- **動画を見ることのメリットを提示**：「この動画を見れば、〜がきっとわかるだろう。」など、動画を見ることで得られる具体的なメリット・答えを明示する。

### 3. 文章構成の王道パターン（冒頭パートの型）
なるべく以下の構成と決まり文句で文章を組み立てること。
- **導入（フック・共感）**:
  「〇〇と聞くと、〜なイメージを持つ人も多いだろう。」など、視聴者の持つ一般的なイメージを挙げて共感から入る。
- **問題提起と視聴者への問いかけ**:
  「だが、〜」と逆接で繋ぎ、意外な事実・悲劇の結末などを提示する。続いて、「果たして、〜一体どうすればよかったのだろうか？」など、**視聴者への問いかけ**を行う。
- **企画紹介（決まり文句）**:
  「今回は、〇〇について議論を繰り広げる、2ちゃんねるの歴史オタクたちを見ていこう。」と定型文を**一字一句違えず**入れる。
- **視聴のメリット（決まり文句）**:
  「この動画を見れば、〇〇がきっとわかるだろう。」という形で**動画を見ることのメリットを提示**して締めくくる。

### 4. 厳密なフォーマット（文字装飾・レイアウト）
- 1行が長くなりすぎないよう、テンポ良く意味の切れ目で改行を入れる。
- 1つのブロック（セル）ごとに、**行頭と行末を必ず `"`（ダブルクォーテーション）で囲う。**
- ダブルクォーテーションの組み全体で、4〜6ブロック程度で収めること。
- 余計な前置きや後書き（「以下のように作成しました」等）は一切出力せず、本文**のみ**を出力すること。

## ゴールデンサンプル（これらと完全に同じトーンで出力すること）

### サンプル1（柴田勝家）
"柴田勝家と聞くと、「鬼柴田」「鬼家老」
そんな勇敢で武勇に優れたイメージを持つ人も多いだろう。"
"だが、信長の家臣として恐れられた彼も、
信長亡き後は豊臣秀吉と敵対し、
賤ヶ岳の戦いで敗れ自害する結末を迎えてしまった。"
"果たして、柴田勝家が秀吉に勝つには
一体どうすればよかったのだろうか？"
"今回は、柴田勝家について議論を繰り広げる、
2ちゃんねるの歴史オタクたちを見ていこう。"
"この動画を見れば、
柴田勝家が秀吉に勝つために必要だったものが
きっとわかるだろう。"
