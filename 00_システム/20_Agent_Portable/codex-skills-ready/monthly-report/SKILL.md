# 月次レポートスキル

## トリガー
- 月末 or ユーザーが月次サマリーを求めたとき

## 手順
1. **売上集計**: 当月のデイリーノートからUber/出前館/YouTube売上を合算
2. **経費取得**: freee会計APIから当月の経費データを取得
3. **損益計算**: 売上 - 経費 = 利益
4. **スプレッドシート更新**: 月次サマリータブに記入
5. **前月比較**: 前月の実績と比較
6. **Discord返信**: レポートを送信
