# 構成パターン集

ネタ（テーマ）と見つかったスレッドの内容に応じて、前半・後半の構成を決定する。
前置きパートを挟んで話題を転換し、視聴者の離脱を防ぐ。

---

## 前置きパートの共通フレームワーク

全パターン共通で、前置きは以下の4要素で構成する。

```
[A] 振り返り：「ここまでは、[前半テーマ]についてのさまざまな議論を見てきたが、」
[B] 転換部 ：新しい疑問・視点・声の紹介（←パターンごとに異なる）
[C] 誘導  ：問いかけ or 共感（「たしかに〜」「一体〜だろうか？」）
[D] 締め  ：「引き続き2ちゃんねるの歴史オタクたちの議論を見ていこう。」
```

[B]の転換部がパターンの核。ここでどう話題を切り替えるかで台本の流れが決まる。

---

## パターン①：人物深掘り型

### いつ使うか
- 前半で「なぜ○○は〜したのか？」系のテーマを議論し、
  後半で「そもそもこの人物はどんな人？」と人物像に掘り下げるとき

### 前半スレの選び方
- 「なぜ〜」「どうして〜」系の議論スレ

### 後半スレの選び方
- 人物論・人物評価・エピソード紹介系のスレ

### 前置きの転換部テンプレート
```
「そもそも○○はどんな人物だったの？」
「とんでもなく優秀なのは知っているけど、人物像がわからない」
と、○○の人物像に疑問を持つ者も現れた。
彼は一体どんな武将だったのだろうか？
```

### 実例
- 前半：毛利元就が天下を狙わなかった理由の議論
- 前置き：「元就はどんな人物だったの？」
- 後半：毛利元就の人物像・エピソード

## パターン②：逆視点型

### いつ使うか
- 前半でA面を議論し、後半で「逆にB面は？」と反対側から切り込むとき
- テーマの裏側・反対概念に関するスレが見つかったとき

### 前半スレの選び方
- ポジティブ側・メジャー側・表側のスレ

### 後半スレの選び方
- ネガティブ側・マイナー側・裏側のスレ

### 前置きの転換部テンプレート
```
「逆に○○は？」「△△はどうだったの？」
など、別の視点から疑問を持つ者も現れた。
たしかに、[裏側のテーマ]はどのようなものだったのだろうか。
```

### 実例
- 前半：戦国時代の豊かな土地について → 後半：貧しかった土地はどこ？
- 前半：47都道府県の代表武将 → 後半：有名な武将がいない県

---

## パターン③：前提を疑う型

### いつ使うか
- 前半で「詳しい人向け」の議論をした後、
  後半で「そもそも〜は本当なの？」と根本をひっくり返すとき
- 豆知識・トリビア系のスレが見つかったとき

### 前半スレの選び方
- 前提を受け入れた上での詳しい議論スレ

### 後半スレの選び方
- 前提自体を疑うスレ、トリビア・雑学系スレ

### 前置きの転換部テンプレート
```
「そもそも○○は本当なの？」
と、疑問を持つ者も現れた。
彼らの意見も大変興味深いものになっている。
```

### 実例
- 前半：神武天皇の年齢や神話の議論 → 後半：存在自体への疑問
- 前半：信長公記の内容の議論 → 後半：なぜ「しんちょうこうき」と読むのか

## パターン④：if仮想型

### いつ使うか
- 前半で結論が出た（or 出そうな）議論の後、
  「もし〜だったら？」と歴史改変シナリオに展開するとき

### 前半スレの選び方
- 議論・考察系スレ（結論が出るもの）
- 「〜はどうすれば勝てた？」系スレ

### 後半スレの選び方
- if系・歴史改変系・「もし〜だったら」系スレ

### 前置きの転換部テンプレート
```
「○○は結局△△だった」という意見が多くを占める一方で、
「もし彼が□□していたら、歴史はどう変わっていたのだろうか？」
と、新たな疑問を持つ者も現れた。
たしかに、彼が□□していればその後の天下の行方は大きく変わっていただろう。
```

### 実例
- 前半：柴田勝家は秀吉に勝てないという議論
- 前置き：「もし勝てていたら歴史はどう変わった？」
- 後半：柴田勝家が勝った場合の歴史改変シナリオ

---

## パターン⑤：再評価型

### いつ使うか
- 前半で批判的な議論・低評価の議論をした後、
  後半で「実は有能だったのでは？」と見直すとき

### 前半スレの選び方
- 批判・ディスり・「〜は無能」系スレ

### 後半スレの選び方
- 擁護・再評価・「実は〜だった」系スレ

### 前置きの転換部テンプレート
```
「実は実力不足ではなく、高く評価されるべき人物もいるのでは？」
「昔は評価が低かったけれど、今は再評価されている武将もいる」
など、手厳しい評価を受けていた武将を再評価する人も多くなってきている。
実は有能だったと言える武将は果たしてどれくらいいるのだろうか。
```

### 実例
- 前半：評価が二つに分かれる武将の批判的議論
- 前置き：「実は再評価されるべき武将もいるのでは？」
- 後半：再評価・擁護の議論

---

## パターン⑥：笑い→ガチ考察型

### いつ使うか
- 前半でネタ・一覧・ランキング等の笑い要素が強いスレを使った後、
  後半で真面目な考察・分析に切り替えるとき
- 出オチ系・打線組んだ系スレが見つかったとき

### 前半スレの選び方
- 一覧系・ランキング系・打線系・ネタスレ
- ツッコミが面白いスレ

### 後半スレの選び方
- 真面目な考察・議論スレ
- 人物評価・歴史分析スレ

### 前置きの転換部テンプレート
```
「○○が多すぎる」問題は実に興味深かったのではないだろうか。
では実際のところ、[本質的な問い]は一体どうだったのか？
ここからは2ちゃんねる民たちによる本気の考察が始まる。
その結論がなかなか面白いものになっている。
```

### 実例
- 前半：「家康が最も恐れた男」一覧の笑い・ツッコミ
- 前置き：「では実際に誰を一番恐れていたのか？」
- 後半：信玄・信長がガチ候補、三英傑比較、家康の再評価

## パターン選定の判断基準

スレッド検索の結果を見て、以下の基準でパターンを選ぶ。

| 見つかったスレの組み合わせ | 推奨パターン |
|---|---|
| 「なぜ〜」系 + 人物評価系 | ①人物深掘り型 |
| 正反対のテーマのスレが2つ以上 | ②逆視点型 |
| 詳しい議論 + トリビア・根本疑問系 | ③前提を疑う型 |
| 結論が出る議論 + if系・改変系 | ④if仮想型 |
| 批判系 + 擁護・再評価系 | ⑤再評価型 |
| ネタ・一覧系 + 真面目な考察系 | ⑥笑い→ガチ考察型 |

複数パターンに当てはまる場合は、**前半と後半の落差が大きい**ものを優先する。
落差が大きいほど前置きでの話題転換が効き、視聴者の離脱を防げる。
