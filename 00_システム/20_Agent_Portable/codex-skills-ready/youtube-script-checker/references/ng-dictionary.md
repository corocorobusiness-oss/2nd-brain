## L2: 差別的トロープパターン（71+正規表現）

### 民族/人種トロープ
```
パターン: 民族名 + 劣等判定動詞
正規表現: (チョン|支那|朝鮮|中国)人?(は|って)(馬鹿|低能|劣等|糞|ゴミ)
重大度: 🔴 高

パターン: 民族名 + 優等主張（逆ヘイト）
正規表現: (日本|日本人|我々)?(だけが|のみが|を除いて)(優秀|優越|正統|正義)
重大度: 🔴 高

パターン: 民族ステレオタイプ主張
正規表現: (〇〇人|朝鮮|シナ)は(必ず|いつも|~が多い)(嘘|裏切|盗み|陰謀)
重大度: 🔴 高
```

### 非人間化パターン
```
パターン: 保護対象グループ + 動物比喩
正規表現: (女|障害者|〇〇人)(は)?(犬|猿|豚|ゴキブリ|害虫|家畜|獣)
重大度: 🔴 高

パターン: 非人間化の相対化
正規表現: (奴ら|やつら|連中)(は)?(人間|知能|感情|尊厳)(じゃない|でない|がない)
重大度: 🔴 高
```

### 女性蔑視トロープ
```
パターン: 女性の物化
正規表現: 女性?(は)?(性的対象|肉便器|金づる|物|商品)
重大度: 🔴 高

パターン: 身体的品評
正規表現: (女|女性)(の)?(胸|脚|顔|容姿)?(を|は)(品定め|採点|評価|値踏み)
重大度: 🟡 中
```

### 障害者ステレオタイプ
```
パターン: 障害の劣等性主張
正規表現: (障害者|盲|聾)(は)(社会|経済|遺伝学的に)(負荷|劣等|無用|排除対象)
```

## L4: 誤検知除外パターン（50+）

### 動詞活用の除外
```
正規表現: [考えた|食べた|来た|できた|やれた|見た|聞いた]
理由: 「えた」「た」を含むがNG表現ではない

正規表現: (渡した|勝った|受けた|脱いた|立った)
理由: 過去形活用の誤検知を避ける
```

### 地名・人名の除外
```
正規表現: (南蛮鎧|蝦夷地|蝦夷族|金玉均|李克用|蒙古斑)
理由: 歴史用語・地政学的固有名詞

正規表現: (武士道|柳生十兵衛|蝦夷|アイヌ)
理由: 歴史的文化的コンテキスト内での使用
```

### 歴史用語の除外
```
正規表現: (バサラ大名|鴉軍|朝鮮出兵|三韓征伐|異民族|異人)
理由: 歴史学的専門用語

正規表現: (処刑|斬首|火あぶり)(の|について|の歴史|時代背景)
理由: 歴史教育的文脈
```

### 否定文脈の除外
```
正規表現: (差別(は|してはならない|すべきでない)|〇〇と呼ぶ(のは|ことは)間違い|使うべきではない)
理由: 教育的・警告的言及
```

### 複合語の除外
```
正規表現: (独眼竜|隻眼の|一つ目|盲目的(に)|(色)盲)
理由: 複合的意味での使用
```

### 2chニックネーム（○○カス型）の除外
```
正規表現: (フロカス|ブリカス|フラカス|アメカス|スカスカ|ダマスカス)
理由: 2ch由来のニックネーム（フロカス＝フロイス、ブリカス＝イギリス、フラカス＝フランス、アメカス＝アメリカ）
      および「カス」を含む一般語（スカスカ、ダマスカス）が「カス」のNGワードに誤検知される
注意: ボロカスは侮辱語として検知対象。FPに入れない。
```

### 死関連の誤検知除外
```
正規表現: (必死|決死|九死|瀕死|致死|生死|起死|活死|死角|死力|死中|死地|死活|死海|死語|死線|死球|死蔵)
理由: 「死」を含むが禁止対象ではない複合語・慣用句
```

### 殺す関連の誤検知除外
```
正規表現: (必殺|殺到|殺風景|殺気|相殺|抹殺)
理由: 「殺」を含むが暴力表現ではない複合語
```

## L5: 辞書外マニュアル検知ルール

### Layer A: 侮辱・蔑称・下品
- くっそ, ショボ, ダサ, ウザ, チンケ, ろくでもない
- 性格終わってる, 性格悪い, 頭おかしい, マジキチ(🔴)
- クズの極み(🔴), ゴミ人間(🔴), 人間以下(🔴)
- 老害, 老害化, 老いぼれ（年齢差別/エイジズム。YouTube保護対象「年齢」に該当）
- 金柑頭, キンカン頭（身体侮辱/ハゲいじり。YouTube保護対象「外見特性」に該当）
- 陰キャ（侮辱）

**リスク**: 🟡 中 - 繰り返し（5回以上）で🔴に昇格。🔴マーク付きは単独で即修正。

### Layer B: 性的・ジェンダー
- 女性物化（「女は〇〇」型ステレオタイプ）
- 好きな女やるで, 孕ませて返却
- 処女性幻想, メス化, 従属女性化表現

**リスク**: 🔴 高 - 単独出現で警告

### Layer C: 民族・国際敏感
- 朝鮮 + 侮辱語コロケーション
- 蝦夷 + 蔑称化（文脈分析）
- 中国人は〜, 韓国人は〜（ステレオタイプ）

**リスク**: 🔴 高 - 即削除候補

### Layer D: YouTube AI特別検知
- 暴力擬音（バキー, ザシュ, ドッドッド）+ 人物指示
- 叫びオンオマトペ（オラオラ, ウリャー）+ 人物攻撃
- やった≠殺したの婉曲表現（文脈で判定）

**リスク**: 🟡 中 - 文脈検査必須

## L8: 保護対象グループ文脈分析ルール

### YouTube 保護対象9属性
1. **人種・民族**: 民族集団, 国籍
2. **宗教**: 宗教グループ
3. **障害**: 身体・精神・知的障害
4. **性自認・性的指向**: トランス, LGBTQコミュニティ
5. **社会的地位**: ホームレス, 移民, 難民
6. **性別**: 女性, 非二元性
7. **年齢**: 児童, 高齢者
8. **医学的状態**: 疾患, 健康状態
9. **外見特性**: 容姿, 身長, 体型

### 危険構文パターン
```
構文: 属性 + 暴力助長
正規表現: (〇〇|属性グループ)(は|を)(殺す|排除|隔離)べき
判定: 🔴 高

構文: 属性 + 憎悪扇動
正規表現: (〇〇|属性)(に|を)対して(憎しみ|怒り|暴力)を(募らせる|扇動)
判定: 🔴 高

構文: 属性 + 非人間化
正規表現: (〇〇|属性)(は)?(人間|人権|尊厳)(でない|がない)
判定: 🔴 高

構文: 属性への身体的侮辱繰り返し
ルール: 同一属性/人物への身体的侮辱・容姿批判が3回以上
判定: 🔴 ハラスメント認定
```

## 検査実行順序

1. **タイトル/メタデータチェック** (L7)
2. **L1直接マッチング** (即座に🔴判定)
3. **L4除外パターン適用** (誤検知除外)
4. **L2-L3正規表現マッチング** (文脈評価)
5. **L5マニュアルルール適用** (特殊ケース)
6. **L6頻度カウント** (累積判定)
7. **L8保護対象グループ分析** (最終判定)

**最終判定**: 🔴は即削除, 🟡は編集要求, ⚪は監視対象
