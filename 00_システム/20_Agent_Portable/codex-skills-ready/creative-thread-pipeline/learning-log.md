# learning-log（creative-thread-pipeline）

各STEPの判断を **"値"でなく"判断ルール/修正パターン"** で記録する（毎回ネタが違うので具体値は汎化しない）。
Phase2 で「同種判断がK回連続承認されたSTEPの自動化提案」に使う（成熟度トラッカー）。**勝手に自動化しない＝提案専用。**

## 形式
`- [STEP] YYYY-MM-DD ルール/パターン（根拠ネタ）`

## 初期シード（平将門・源義経から確立した型）
- [STEP3] 解説対象＝**その語を知らないと議論が追えない難読固有名トップN**（平将門/源義経）。
- [STEP3] 解説の挿入位置＝**対象用語クラスタの末尾（次の話題へ移る直前）**。初出直後ではない（平将門：大蔵省は8初出だが解説は15直後）。
- [STEP3] 既定の型＝解説 各スレ2本(計4)・各2〜4セル／冒頭5〜6・前置き3〜4・締め5〜6セル／>>1バースト概ね seq2〜5。すべて「既定＋増減提案→確認」。
- [STEP4] 大の判定＝**長文（≒88字+）を大**。境界±5字は提案→確認。大は返信でも下/>>を付けない（大優先）。
- [STEP4] 返信先は**会話の流れ準拠**。後方ジャンプ（独立トピック完結後に前へ戻る＝①66型）は直近 or 同一wallへ寄せる（機械補正でなくLLM判断）。
- [STEP4.5] must_fix の常連＝**講義臭（元号+西暦+終止形のナレVO化）／二重主語/文末表現の文頭（「知らんけど」文頭）／現代実在個人の実名**。掲示板口調へ・述語一本化・文末へ・匿名化。
- [STEP5] 近代政治色（GHQ/米軍/進駐軍 等）→**歴史的記述へ一般化**（戦後の区画整理/重機/国家事業/お役所）。
- [全般] 本文を改変する広域修正の後は **大/下再判定→反転レスの返信マップ再判断→差分再qa** を1セットで回す。
