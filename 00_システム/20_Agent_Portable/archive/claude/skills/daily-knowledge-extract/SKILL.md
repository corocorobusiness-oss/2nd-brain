# デイリーノート → 知識 自動昇華

その日のデイリーノートを読み、「ただのメモ」で埋もれている価値ある情報を
知識ベースに昇格させて"勝手に育つ第2の脳"を実現する。

## 対象・保存先（2nd-Brain/CLAUDE.md準拠）
- 読む: `05_日誌/YYYY-MM-DD.md`（当日。前日も未処理なら拾う）
- 保存先カテゴリ:
  - アイデア/企画 → `03_知識ベース/YouTube・コンテンツ制作/` 等 該当サブ
  - スキル・ノウハウ/学び → `03_知識ベース/学習メモ/`
  - 人物・関係 → `03_知識ベース/人の情報（relationship）/`
  - ツール・技術 → `03_知識ベース/テクノロジー・ツール/`
  - ビジネス・マーケ → `03_知識ベース/ビジネス・マーケティング/`

## 手順
1. 当日（必要なら前日）の日誌を読む
2. 「思いつきメモ/Inbox」「思考と感情のログ」「抽出された情報」セクション等から、
   **再利用価値のある情報**を抽出（売上数字・タスク・日常の愚痴は除外）
3. 各情報を上記カテゴリに分類
4. 知識ベースに追記/新規作成：
   - 既存ノートに該当があれば**追記**（重複作成しない）
   - 無ければ新規。ファイル冒頭に出典「(YYYY-MM-DD 日誌より)」
   - 関連する既存ノート/メモリに `[[リンク]]` を張る
5. 昇華した分は日誌の該当箇所に `✅昇華済` を付ける（二重処理防止）
6. Discord #メモ (chat_id 1512911464439087295) に簡潔報告：
   「📥 今日の日誌から N件を知識化：・○○→学習メモ ・△△→…」

## 安全ルール
- **追記中心**。既存の知識を削除・上書きしない（統合はknowledge-gardenerが担当）
- 価値が曖昧なものは昇華せず日誌に残す（取りこぼしの方がマシ）
- 不可逆操作なし
