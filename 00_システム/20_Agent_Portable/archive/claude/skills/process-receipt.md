# レシート処理スキル

## トリガー
- Discordでレシート画像が送られたとき

## 手順
1. **画像ダウンロード**: Discord添付ファイルをダウンロード
2. **画像処理**: `~/.claude/scripts/enhance-receipt.py` でコントラスト・シャープネス強化
3. **読み取り**: 強化画像から日付・店名・金額・支払方法を読み取り
4. **Google Drive保存**: `経費精算/YYYY/MM/YYYYMMDD_取引先名_金額.jpg`
5. **支払方法判定**:
   - 現金: freeeファイルボックスにアップロード → freee取引登録（車両費/事業主借）
   - クレジットカード: Drive保存のみ
6. **スプレッドシート更新**: キャッシュフロータブにガソリン代を追加
7. **Discord返信**: 処理結果を報告
