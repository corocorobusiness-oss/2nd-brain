# 解説パート作成ガイド

## 目的
ユーザーから提供された歴史上の名将、組織、合戦、地名などの「テーマ」や「コメント」をもとに、YouTube台本の「解説パート」を**ゼロから書き起こす**。過去に作成された台本と**完全に同一のフォーマット・トーン**で新規出力すること。

## 挿入タイミングと目的
- **狙い**：視聴者が「学び」を感じる教育的価値を高め、YouTube AIに「価値あるオリジナルコンテンツ」と認識させるため。
- **挿入箇所と数**：1本の台本に必ず「2個以上の解説」を挿入する。（冒頭から前置きの間に１本、前置きから締めまでの間に１本は必ず入れる）
- **対象箇所**：全てのコメントに解説を挟むと会話のテンポが崩れるため、視聴者の理解を深めるために「説明があったほうがわかりやすいコメント（予備知識がないと理解が難しい用語など）」に対して解説の追加を検討すること。
- **解説するカテゴリ**：人物 / 歴史的事柄 / 歴史用語 / 過去の地域名 / 時代 など

## 出力ルール・制約

AIは、以下の厳密なルールに従って自身の知識を元に**ゼロから文章を生成**し、過去の原稿と見分けがつかないレベルで完全にトーンとフォーマットを統一すること。

### 1. 文字数と文章構成の王道パターン
- **文字数**：100〜250文字で作成すること。
- 必ず以下の構成で文章を組み立てること。
- **導入**:
  「『〜（解説対象の名前）とは、』」で書き出し、「〜時代に〜を拠点とした武将である。」のように、事実・定義を客観的に端的に述べる。
- **中盤（実績・特徴・経過）**:
  「〜で知られ、」「〜という形で名を残している。」「特に〜で〜したことで知られている。」など、感情的にならず史実を淡々と並べる。
- **結末（結果・歴史的評価）**:
  「最終的に〜滅ぼされた。」「結果的に〜土台を作った人物でもある。」「大きな転機となったのである。」など、最終的な結末や後世の評価で締めくくる。

### 2. 厳密なフォーマット（文字装飾・レイアウト）
- 1行が長くなりすぎないよう、テンポ良く意味の切れ目で改行を入れる。
- 冒頭文同様、**2～3行ずつでセルを分割（ブロック化）**すること。
- そのブロックごとに、**行頭と行末を必ず `"`（ダブルクォーテーション）で囲う。**
- ダブルクォーテーションの組みは、全体で**3〜6ブロック程度**に簡潔にまとめること。
- 余計な前置きや後書き（「以下のように作成しました」等）は一切出力せず、本文**のみ**を出力すること。

### サンプル1（彦火火出見尊）
"彦火火出見尊とは、
日本神話に登場する皇祖神の一柱で、
山幸彦としても知られる人物である。"
"海神の娘・豊玉姫と結ばれ、
その子が鵜葺草葺不合尊、
孫にあたるのが"
"初代天皇とされる神武天皇である。
海幸彦との釣針の物語や、
海宮訪問譚などで知られ、"
"天皇家の神話的祖系をつなぐ
重要な存在とされている。"
