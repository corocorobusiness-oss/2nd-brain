# 週次Gmail整理タスク

以下の手順でGmailの整理を行い、結果をDiscordに報告してください。

## 1. Gmailチェック
Gmail MCPを使って全メールを検索してください。

## 2. 不要メール判別基準

### 自動削除対象（不要）:
- セキュリティ通知（Google, Apple, Discordのログイン通知）
- ウェルカム・案内メール（サービスの初回登録メール）
- 設定完了・処理済み通知
- プロモーション・マーケティングメール
- Apps Scriptエラー通知
- アカウント設定確認メール

### 残す（重要）:
- 領収書・レシート（Anthropic, その他サービス）
- AdSense入金履歴・支払い関連
- 購入履歴（Brain等）
- YouTube/事業関連メール
- 経費関連メール
- マネーフォワード関連の重要通知
- 税務関連メール

## 3. 削除実行
Playwrightブラウザ操作でGmailを開き、不要メールを選択して削除してください。
- mail.google.com にアクセス
- 不要メールのチェックボックスを選択
- 削除ボタンを押す

## 4. Discord報告
Discord チャンネル 1486946641389817899 に以下を報告：
- 削除したメール件数と内容サマリー
- 残したメール件数
- 判断に迷ったメールがあれば確認依頼
