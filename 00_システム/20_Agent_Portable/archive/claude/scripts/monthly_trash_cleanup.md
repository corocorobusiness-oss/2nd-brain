# 月次ゴミ箱削除タスク

以下の手順でMacのゴミ箱とGmailのゴミ箱を空にし、結果をDiscordに報告してください。

## 1. Macのゴミ箱を空にする
```
rm -rf ~/.Trash/*
```

## 2. Gmailのゴミ箱を空にする
Playwrightブラウザ操作で：
1. mail.google.com にアクセス
2. 左サイドバーから「ゴミ箱」を開く
3. 「ゴミ箱を今すぐ空にする」をクリック
4. 確認ダイアログで承認
