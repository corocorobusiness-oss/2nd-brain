# skill-updates — スキル正本（~/agent-skills）への反映待ち更新パッケージ

リモートセッション（Claude Code on the web など）はMac miniの `~/agent-skills`（スキル正本）に
直接触れないため、スキル更新はここに完全な形で置き、ローカルのエージェントが正本へ反映する。

## ライフサイクル

1. リモートが更新パッケージ（スキル1個分の完全なディレクトリ＋INSTALL.md）をここへ置く
2. `06_エージェント運用/20_実行キュー/今日やる.md` に反映タスクを積む
3. ローカルエージェント（Codex / Claude Code）が INSTALL.md の手順で `~/agent-skills` へ反映
   - 反映前に正本側を必ずバックアップ（正本はgit管理なのでコミットでも可）
   - 同梱テストがある場合は反映後にPASSを確認する
4. 反映完了後、パッケージを `99_アーカイブ/skill-updates/` へ移動し、作業ログに記録する
   （AIに削除権限なし・風神さん流。アーカイブ移動はOK）

## 注意

- ここは**一時置き場**。スキルの編集正本は常に `~/agent-skills`
- `codex-skills-ready/`（FROZENスナップショット）や `live-backups/` を編集正本と混同しない
