# 締めパート作成ガイド

## 目的
ユーザーから提供された「動画全体のテーマ」や「議論の主な内容」をもとに、YouTube台本の最後を飾る「締めパート」を**ゼロから書き起こす**。過去に作成された台本と**完全に同一のフォーマット・トーン**で新規出力すること。

## 出力ルール・制約

AIは、以下の厳密なルールに従って**ゼロから文章を生成**し、過去の原稿の王道パターンと見分けがつかないレベルで完全にトーンとフォーマットを統一すること。

### 1. 文字数とセル分割のルール
- 締めの文は**最大280文字**で作成すること。
- 締めの文章は切りのいい箇所で**2～3行ずつ**セルを分け（ブロック化し）ること。

### 2. 必須要素（視聴者を行動させること）
締めの文は以下の構成で作成すること。**【生成パート】と【固定パート】を明確に区別すること。**

#### 【生成パート】（テーマに合わせて毎回ゼロから書く）
以下の要素を**この順番で**含めること：
1. **視聴者への問いかけ**：「いかがだっただろうか？」など。
2. **動画の要約と所感**：動画の内容を簡単にまとめ、制作側の意見を述べる。（例：「2ちゃんねるの歴史オタクたちの議論を見ていると、〜と思えた。」）

#### 【固定パート】（毎回そのまま使う。一字一句変えないこと）
生成パートの後に、以下の2ブロックを**そのままコピーして**末尾に付ける：

```
"今回の動画について他の意見や
感想があったらぜひコメントで教えてね！"
"他にも歴史にまつわる情報をまとめた
動画が満載なので、ぜひ見てみてくださいね。
ご視聴ありがとうございました！"
```

**CRITICAL**: この2ブロックは固定テンプレートであり、テーマに合わせて変更してはならない。必ずこの文言をそのまま出力すること。

### 3. 厳密なフォーマット（文字装飾・レイアウト）
- 1行目の「本日はここまで。」のみ、ダブルクォーテーションで囲わない。
- それ以降の2〜3行ずつの意味の塊（セル）ごとに、**行頭と行末を必ず `"`（ダブルクォーテーション）で囲う。**
- 余計な前置きや後書き（「以下のように作成しました」等）は一切出力せず、本文**のみ**を出力すること。
