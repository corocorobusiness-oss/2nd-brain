# 打線表記の読み方ルール

2ch歴史系台本で使われる打線（野球のスタメン）表記の固定読みルール。

## 基本ルール

「◯（ポジション漢字）」→「◯ばん＋ポジション名のカタカナ読み」

## 一覧

| 表記 | 読み | 備考 |
|---|---|---|
| 1（遊） | 1ばんしょーと | ショート |
| 2（二） | 2ばんせかんど | セカンド |
| 3（三） | 3ばんさーど | サード |
| 4（捕） | 4ばんきゃっちゃー | キャッチャー |
| 5（中） | 5ばんせんたー | センター |
| 6（左） | 6ばんれふと | レフト |
| 7（右） | 7ばんらいと | ライト |
| 8（一） | 8ばんふぁーすと | ファースト |
| 9（投） | 9ばんぴっちゃー | ピッチャー |
| （DH） | でぃーえいち | 指名打者 |
| （指） | でぃーえいち | 指名打者（漢字表記） |

## 注意事項

- 打順の数字は台本によって変わるが、ポジションの読みは固定
- 打順とポジションの組み合わせが異なる場合もある（例：1（中）= 1ばんせんたー）
- 台本に打線が含まれているかどうかはユーザーが指定するか、文脈から判断する
