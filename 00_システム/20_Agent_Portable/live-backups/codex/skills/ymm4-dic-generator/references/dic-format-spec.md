# YMM4 単語辞書(.dic)フォーマット仕様

## 基本構造

タブ区切りテキストファイル。拡張子は `.dic`。

## カラム定義

```
カラム1(TAB)カラム2(TAB)カラム3(TAB)カラム4(TAB)カラム5(TAB)カラム6
```

| カラム | YMM4上の名称 | 内容 | 値 |
|---|---|---|---|
| 1 | 有効 | 有効フラグ | `1`（固定） |
| 2 | 正規表現 | 正規表現マッチ | `0`（通常）または `1`（正規表現有効） |
| 3 | 変換前 | 単語（表層形） | 漢字やカタカナの表記 |
| 4 | 変換後 | 読み | ひらがな |
| 5 | 備考 | 空欄 | （空文字列） |
| 6 | - | フラグ | `1`（統一） |

## 正規表現フラグ（カラム2）について

- 通常のエントリ：`0`（リテラルマッチ）
- 打線エントリなど半角数字＋全角括弧を含むもの：`1`（正規表現マッチ）

CRITICAL: 半角数字と全角括弧（）を含むエントリ（例：`2（二）`）は、
カラム2を `1` にしないとYMM4がマッチしない。

## エントリ例

通常エントリ：
```
1	0	伊達政宗	だてまさむね		1
1	0	政宗	まさむね		1
1	0	独眼竜	どくがんりゅう		1
```

打線エントリ（正規表現有効）：
```
1	1	1（遊）	1ばんしょーと		1
1	1	2（二）	2ばんせかんど		1
1	1	（DH）	でぃーえいち		1
```

## 運用方針

- **毎回書き換え方式**：台本ごとにその台本の用語だけでファイルを新規作成する
- 前の台本のエントリは上書きされる
- エンコーディング: UTF-8
- 改行コード: LF
