## 変換必須パターン

| パターン | 変換先の方針 | 例 |
|---|---|---|
| 動物蔑称（猿/犬で人を呼ぶ） | 歴史呼称に変換 | 猿→太閤、犬→大御所 |
| 伏字（〇○◯●■□） | 全て言い換えに完全変換 | ガ〇ジ→変人 |
| ジェンダーステレオタイプ | 個人名に変換 | 「女性は〇〇」→「愛姫は〇〇」 |
| 女性物化表現 | 中立表現に変換 | 「女やるで」→「嫁やるで」 |
| 身体侮辱 | 中立表現に変換 | デブ→恰幅良い男 |
| 「性格終わってる」型 | マイルド表現に | 性格終わってる→クセ強い |
| くっそ＋形容詞 | めちゃくちゃ＋形容詞 | くっそ有能→めちゃくちゃ有能 |
