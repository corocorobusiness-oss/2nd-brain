# Codex用 知識資産フォルダ

Codexアプリ（OpenAI）で使うAGENTS.mdテンプレートとプロンプトテンプレートをまとめている。

## ファイル一覧

| ファイル | 用途 |
|---------|------|
| `AGENTS_global.md` | グローバル設定（全プロジェクト共通のルール） |
| `AGENTS_youtube_script.md` | YouTube台本制作用AGENTS.md |
| `AGENTS_research.md` | リサーチ・調査用AGENTS.md |
| `AGENTS_brainstorm.md` | 壁打ち・アイデア整理用AGENTS.md |
| `prompt_template_5elements.md` | 5要素プロンプトテンプレート（汎用） |

## 使い方

### AGENTS.mdをCodexプロジェクトに設定する
1. Codexのプロジェクトフォルダ内に `AGENTS.md` というファイルを作る
2. このフォルダの該当テンプレートをコピペして貼り付ける
3. 必要に応じて中身をカスタマイズする

### グローバルAGENTS.mdの場所（Mac）
`~/Library/Application Support/Codex/AGENTS.md`（または設定画面から確認）
