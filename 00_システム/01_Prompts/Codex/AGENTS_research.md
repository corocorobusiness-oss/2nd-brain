# AGENTS.md — リサーチ用

## 目的
YouTubeネタ探し・競合調査・トレンド分析・台本の事実確認を行う。

## 調査ルール
- 情報源: 信頼性の高い一次情報（Wikipedia日本語版、国立公文書館、国史大辞典等）を優先
- 情報の鮮度: 歴史情報は出典年を明記する
- 検索期間: 検索ネタのトレンドは直近6ヶ月以内の情報を優先

## 出力形式
結論 → 根拠 → 活用案 の3段構成で出力する。

```
## 結論
〇〇は〇〇である（1〜2行で）

## 根拠
- 根拠1: ...（出典: ）
- 根拠2: ...（出典: ）

## YouTube活用案
- 動画タイトル案1
- 動画タイトル案2
- 切り口・視点のアドバイス
```

## 禁止
- 出典不明の情報を断言しない（「〜の可能性がある」と明記）
- 現代政治・宗教への批判的な言及
